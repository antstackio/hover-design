# Select

### Quick start

Here's a quick start guide to get started with the Select component

### Importing Component

```jsx
import { Select } from "@hover-design/react";
```

### Code Snippets and Examples

import { Select, Loader } from "@hover-design/react";
import { useState } from "react";

##### Controlled Select

```jsx
const [selectedValue, setSelectedValue] = useState("first");

const options = [
  { label: "First", value: "first" },
  { label: "Second", value: "second", disabled: true },
  { label: "Third", value: "third" },
  { label: "Fourth", value: "fourth" },
];

<div className="App">
  <Select
    value={selectedValue}
    onChange={(value) => setSelectedValue(value)}
    options={options}
  />
</div>;
```

export const App = () => {
  const [selectedValue, setSelectedValue] = useState("first");
  const options = [
    { label: "First", value: "first" },
    { label: "Second", value: "second", disabled: true },
    { label: "Third", value: "third" },
    { label: "Fourth", value: "fourth" },
  ];
  return (
    <Select
      width="40%"
      value={selectedValue}
      onChange={(value) => setSelectedValue(value)}
      options={options}
    />
  );
};

<App />

### Option Object

| Key      |                type                | Optional ? |
| :------- | :--------------------------------: | ---------: |
| label    |              `string`              |         No |
| value    |      `string` &#124; `number`      |         No |
| disabled |             `boolean`              |        Yes |
| ref      | `MutableRefObject<HTMLDivElement>` |        Yes |

### Loading Option

| Key            |    type     | Optional ? |
| :------------- | :---------: | ---------: |
| loadingContent | `ReactNode` |        Yes |
| loader         | `ReactNode` |        Yes |

### Keyboard Controls

> Keyboard controls will only work when the ** Select ** is focused (selected)

| Key       |                           type                            |
| :-------- | :-------------------------------------------------------: |
| Space     |                   `Opens Options List`                    |
| ArrowDown |                       `Next Option`                       |
| ArrowUp   |                     `Previous Option`                     |
| Home      |      _ if not **disabled ** _ <br/> `First Option `       |
| End       |       _ if not **disabled ** _ <br/> `Last Option`        |
| Backspace | _ if **isMulti ** _ <br/> `Clears the last selected pill` |
