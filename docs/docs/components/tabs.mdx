# Tabs

### Quick start

Here's a quick start guide to get started with the Tabs component

### Importing Component

```jsx
import { Tabs } from "@hover-design/react";
```

### Code Snippets and Examples

import { Tabs } from "@hover-design/react";
import { useState } from "react";
import { FiArrowRightCircle } from "react-icons/fi";
import { FiCpu } from "react-icons/fi";

##### Tabs Default

```jsx
const tabsData = [
  {
    label: "First",
    value: "first",
  },
  {
    label: "Second",
    value: "second",
    disabled: true,
  },
  {
    label: "Third",
    value: "third",
    icon: <FiArrowRightCircle />,
  },
  {
    label: "Fourth",
    value: "fourth",
    icon: <FiCpu />,
    badge: "13",
  },
];

<div className="App">
  <Tabs defaultValue={"first"} tabsData={tabsData}>
    {(selectedTab) => <div>{selectedTab.label}</div>}
  </Tabs>
</div>;
```

> **_(selectedTab)_** contains the whole tab object which is selected.

##### Controlled Tabs

```jsx
const [activeTab, setActiveTab] = useState("first");

const tabsData = [
  {
    label: "First",
    value: "first",
  },
  {
    label: "Second",
    value: "second",
    disabled: true,
  },
  {
    label: "Third",
    value: "third",
    icon: <FiArrowRightCircle />,
  },
  {
    label: "Fourth",
    value: "fourth",
    icon: <FiCpu />,
    badge: "15",
  },
];

<div className="App">
  <Tabs
    value={activeTab}
    onChange={(selectedTab) => setActiveTab(selectedTab.value)}
    value
    tabsData={tabsData}
  >
    {(selectedTab) => <div>{selectedTab.label}</div>}
  </Tabs>
</div>;
```

> **_(tabsData)_** contains the whole tab object which is selected.

export const App = () => {
  const [activeTab, setActiveTab] = useState("first");
  return (
    <Tabs
      grow
      value={activeTab}
      onChange={(selectedTab) => setActiveTab(selectedTab.value)}
      tabsData={[
        {
          label: "First",
          value: "first",

        {

          icon: <FiCpu />,
          badge: "13",
        },
      ]}
    >
      {(selectedTab) => <div>{selectedTab?.label}</div>}
    </Tabs>
  );
};

### Props Reference

| Attributes      |                                      Values                                       | Default Value | Optional ? |
| :-------------- | :-------------------------------------------------------------------------------: | :-----------: | ---------: |
| defaultValue    | _ must be equal to value from ** Tabs Object ** _ <br/> `string` &#124; `number`  |               |        Yes |
| value           | _ must be equal to value from ** Tabs Object ** _ <br/> `string` &#124; `number`  |               |        Yes |
| orientation     |                          `horizontal` &#124; `vertical`                           | `horizontal`  |        Yes |
| variant         |                     `default` &#124; `outline` &#124; `pill`                      |   `default`   |        Yes |
| color           |                                     `string`                                      |   `#2F80ED`   |        Yes |
| backgroundColor |                                     `string`                                      |   `#d7e9ff`   |        Yes |
| outlineColor    |                                     `string`                                      |   `#EBECF0`   |        Yes |
| textAlign       |                       `left` &#124; `center`&#124; `right`                        |   `center`    |        Yes |
| roundness       |                                     `string`                                      |     `4px`     |        Yes |
| onChange        |                            `(selectedTabObject)=>void`                            |   `()=>{}`    |        Yes |
| grow            |                       grows to fit the containter `boolean`                       |    `false`    |        Yes |
| size            | _ **height** for ** horizontal ** & **width** for ** vertical ** _ <br/> `string` |    `40px`     |        Yes |
| tabsData        |                           `Array of` **_Tabs Object_**                            |               |         No |
| children        |  _ not a prop but children to the `<Tabs/>`_ <br/> `(selectedTabObject)=> JSX `   |               |         No |

### Tabs Object

| Key      |                type                | Optional ? |
| :------- | :--------------------------------: | ---------: |
| label    |              `string`              |         No |
| value    |      `string` &#124; `number`      |         No |
| icon     |           `JSX Element`            |        Yes |
| disabled |             `boolean`              |        Yes |
| badge    |   `string` &#124; `JSX Element`    |        Yes |
| ref      | `MutableRefObject<HTMLDivElement>` |        Yes |

### Keyboard Controls

> Keyboard controls will only work when the ** Tab ** is focused (selected)

| Key        |                    type                     | Orientation |
| :--------- | :-----------------------------------------: | ----------: |
| ArrowRight |                 `Next Tab`                  |  horizontal |
| ArrowLeft  |               `Previous Tab`                |  horizontal |
| ArrowDown  |                 `Next Tab`                  |    vertical |
| ArrowUp    |               `Previous Tab`                |    vertical |
| Home       | _ if not **disabled ** _ <br/> `First Tab ` |        both |
| End        |  _ if not **disabled ** _ <br/> `Last Tab`  |        both |
