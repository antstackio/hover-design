# Flex

## Quick start

Here's a quick start guide to get started with the flex component

## Importing flex

```jsx
import { Flex } from "@hover-design/react";
```

## Code Snippets and Examples

## Props Referece

| Attributes     |                                                                                     Values                                                                                     | Optional ? |
| :------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------: | ---------: |
| flexDirection  |                                                       `row` &#124; `column` &#124; `row-reverse` &#124; `column-reverse`                                                       |        Yes |
| flexWrap       |                                                                            `wrap` &#124; `no-wrap`                                                                             |        Yes |
| alignContent   | `normal` &#124; `start` &#124; `center` &#124; `end` &#124; `flex-start` &#124; `flex-end` &#124; `space-between` &#124; `space-around` &#124; `space-evenly` &#124; `stretch` |        Yes |
| justifyContent |            `normal` &#124; `start` &#124; `center` &#124; `end` &#124; `flex-start` &#124; `flex-end` &#124; `space-between` &#124; `space-around` &#124; `stretch`            |        Yes |
| alignItems     |            `normal` &#124; `start` &#124; `center` &#124; `flex-start` &#124; `flex-end` &#124; `space-between` &#124; `space-around` &#124; `stretch` &#124; `end`            |        Yes |
| flexShrink     |                                                                                    `number`                                                                                    |        Yes |
| flexGrow       |                                                                                    `number`                                                                                    |        Yes |
| flexBasis      |                                            `width` &#124; `max-content` &#124; `min-content` &#124; `fit-content` &#124; `content`                                             |        Yes |
| gap            |                                                                  `normal` &#124; `length` &#124; `percentage`                                                                  |        Yes |
| onClick        |                                                                                 `() => void;`                                                                                  |        Yes |
| onHover        |                                                                                 `() => void;`                                                                                  |        Yes |
| display        |                                                                          `flex` &#124; `inline-flex`                                                                           |        Yes |
