# Grid

## Quick start

Here's a quick start guide to get started with the Grid component

## Importing Grid

```jsx
import { Grid } from "@hover-design/react";
```

## Code Snippets and Examples

### Grid Default

import { Grid } from "@hover-design/react";
import { Card } from "@hover-design/react";

```jsx
<Grid>
  <div>Grid 1</div>
  <div>Grid 2</div>
</Grid>
```

<Grid>
  <Card variant="outline">Grid 1</Card>
  <Card variant="outline">Grid 2</Card>
</Grid>

### Grid Template Columns

```jsx
<Grid columnGap="12px" gridTemplateColumns="200px 200px 200px">
  <div>Grid 1</div>
  <div>Grid 2</div>
  <div>Grid 3</div>
  <div>Grid 4</div>
  <div>Grid 5</div>
</Grid>
```

<Card variant="outline">
  <Grid columnGap="12px" gridTemplateColumns="200px 200px 200px">
    <Card variant="outline">Grid 1</Card>
    <Card variant="outline">Grid 2</Card>
    <Card variant="outline">Grid 3</Card>
    <Card variant="outline">Grid 4</Card>
    <Card variant="outline">Grid 5</Card>
  </Grid>
</Card>

### Grid Template Rows

```jsx
<Grid gap="12px" gridTemplateRows="200px 200px 200px">
  <div>Grid 1</div>
  <div>Grid 2</div>
  <div>Grid 3</div>
  <div>Grid 4</div>
  <div>Grid 5</div>
</Grid>
```

<Card variant="outline">
  <Grid gap="12px" gridTemplateRows="200px 200px 200px">
    <Card variant="outline">Grid 1</Card>
    <Card variant="outline">Grid 2</Card>
    <Card variant="outline">Grid 3</Card>
    <Card variant="outline">Grid 4</Card>
    <Card variant="outline">Grid 5</Card>
  </Grid>
</Card>

### Other properties

```jsx
<Grid
  columnGap="12px"
  gridAutoRows="100px"
  gridTemplateColumns="repeat(3, 1fr)"
  gap="20px"
>
  <div>Grid 1</div>
  <div>Grid 2</div>
  <div>Grid 3</div>
  <div>Grid 4</div>
  <div>Grid 5</div>
  <div>Grid 6</div>
  <div>Grid 7</div>
</Grid>
```
